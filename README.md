Exercises

1. Profit Margin Calculator
   - It calculates profit and profit margin based on revenue and cost that is prompted with 2 decimal places.
Assumptions:
- Revenue and cost are entered as positive numbers.
- User puts in floats or integers.

2. Credit Score Evaluator
   - It categorizes loan eligibility based on credit scores using conditionals.
Assumption:
- Input is an integer between 300 and 850.
- Scores outside this range are "Invalid"

3. Customer Greeting
  - Cleans and fromats customer names while handling extra whitespace, Capatilizes names correctly, and extracts the first name to create a personalized greeting.
Assumptions:
- If user enters a name, the code only uses the first name for the greeting.
- if the name field is left empty, the system defaults to "Valued Customer."

4. Tax Bracket Determiner
   - Determines an employee's tax racket based on their annual income. It returns low,medium or high (a specified bracket) and calculates estimated tax due.
   Assumptions:
- Income is provided as a float that's not negative.
- Tax rates are fixed at 10, 20 and 30% based on defined income thresholds.

5. Product Category Matcher
   - Uses a match-case statement to categorize products into margin tiers based on their name. It converts text to lowercase and strips extra whitespaces.
   Assumptions:
- The product category is determined by specific key words.
- Any product not matching the predefined keywords is marked as "Uncategorized."
